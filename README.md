## Launch kit (2026-07-09)

The launch-pack layers added on top of the research corpus. Start at the reading room; it walks every artifact in narrative order (Vision, Pitch, Technical, Financial, Legal, plus an internal meeting-pack shelf) with a built-vs-designed status per entry.

| Folder / file | Contents |
|---------------|----------|

| `07-roadmap/` | Execution map: 10 workstreams, dependency spine, `ROADMAP.md`, and per-workstream backlogs (W2 entity/equity, W8 unit-economics/ICP, W9 data-governance feed the Financial and Legal lanes). |
| `08-brand/` | Brand package: naming and identity candidates only (never a chosen name), each with a same-day knockout screen. `source.md` + `index.html`. |

| `09-deliverables/` | The packaged deliverables (7 folders, each `source.md` + `index.html`): `founder-memo/`, `one-pager/`, `exec-overview/`, `financials-icp/`, `legal-readiness/`, `pitch-deck/`, `whitepaper/`.
